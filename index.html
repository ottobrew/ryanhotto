Delivery Routing Program

This is a Delivery Routing Program written in Python 3 using PyCharm 2022.2.2 (Community Edition) IDE.

A project for Algorithms & Data Structures II course, this program utilizes a hash table and greedy algorithm to efficiently route 40 packages for delivery by two trucks in fewer than 140 miles, while meeting detailed requirements for truck space limits, individual package delays, delivery deadlines, and one address change. CLI allows user to track packages by package id and check delivery status of packages by entering a time.



Scheduling App

Scheduling Application developed with Java, JavaFX, and SQL. Built for the purpose of creating, viewing, tracking, and editing appointments and customers for a variety of business applications. Application connects using API to MySQL database. User authenticates with username and password. Front-end UI allows user to add, update, and delete items from database.

* IntelliJ IDEA 2021.1.3 Community * Java SDK v.11.0.13 * Javafx-sdk-17.0.1 * Mysql-connector-java-8.0.27

Synopsis: To run the program, begin by launching the app. Login with username (test) and password (test). Once login is validated, you will receive a notification of any upcoming appointments. You will then arrive at the main landing page which shows complete tables of Appointments and Customers, along with buttons to Add, Update, or Delete selected appointments or customers. You can also go to a Reports page by clicking the Reports button at the bottom. By clicking on the Add Appointment or Add Customer button, you will be taken to respective forms in which you can add details and then save records to the database. On the main landing page, you also have the option to select from the tableviews an appointment or customer. Once selected, you are able to update individual details or delete records by using the buttons on the right pane. (WGU Software II project)
